◆ isHit()

bool Game::isHit ( int  tRow,
int  tCol,
Grid *  tGrid 
)

verifies that player's shot inputs are valid, then checks if the shot was a hit or miss.

This function first makes sure that tRow and tCol are within the boundaries of the board, then makes sure that the board location at (tCol, tRow) has not been fired at already. If the inputs are valid, the function then calls Grid's setValue function to assign H (if it's a hit) or M (if it's a miss) at (tCol, tRow).

Precondition
none
Postcondition
tGrid's m_grid[tRow][tCol] is updated to H if hit, M if miss.
Returns
true if hit, false if miss.
Parameters
tRowthe target row the player is firing at.
tColthe target column the player is firing at.
tGridthe game board being fired at.
